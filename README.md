# Databasteknik – Labb 3

This project is part of **Labb 3** in the course **Databasteknik**.  
It is a PHP web application connected to a relational database with classical music data.

## Content

- **Database tables**
  - CD, Composer, Composition, comp_CD, Orchester, orch_cd  
  (shown via `tabell1.php` – `tabell6.php`)

- **Predefined queries**
  - List harpsichord pieces
  - List concertos
  - CDs containing BWV 780
  - Glenn Gould recordings
  - Number of recordings of the same piece
  - Compositions in F♯ minor

- **Search forms**
  - Search by BWV number
  - Search by instrument

## Requirements

- PHP
- MySQL / MariaDB
- Web server (e.g. Apache / XAMPP)

## How to Run

1. Place the project in the web server root
2. Import the database
3. Open `index.html` in a browser
